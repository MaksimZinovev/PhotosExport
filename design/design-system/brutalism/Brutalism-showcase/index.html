FAQ

Questions, answered.

Is this a Figma library, a code library, or both?

Both. Tokens flow from one source of truth into Figma styles and into the codegen pipeline at the same time.

Can we ship our own brand theme?

Yes — fork the token file, change the palette and type stack, and every component reskins automatically.

What about accessibility?

Color contrast meets WCAG AA on every surface. Components ship with focus rings, ARIA roles, and keyboard handling.

How do you handle dark mode?

Every token has a paired dark value. The system flips at the document level — no per-component overrides needed.
